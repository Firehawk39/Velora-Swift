# User Listening Preferences

This document serves as the long-term textual memory for the user's explicit and inferred musical tastes. 
The AI Engine (FastAPI) will append notes to this file based on conversational interactions.

## Explicit Preferences
*(Manually stated by the user during chat)*
- Example: "The user mentioned they hate overly synthesized pop from the early 2000s."

## Inferred Habits
*(Determined from telemetry and listening history)*
- Example: "Tends to play high BPM electronic music during the morning (08:00 - 11:00)."
- Example: "Frequently skips acoustic tracks when listening to the 'Focus' playlist."
